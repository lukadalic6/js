# JavaScript — Test vježba (Predavanja 1, 2, 3)

**Web dizajn · Samostalni zadatak**

---

## Upute

1. Kloniraj ovaj repo:
   ```bash
   git clone [URL]
   cd js-test-vjezba
   ```

2. Otvori `script.js` u VSC-u

3. Riješi sve zadatke — svaki zadatak je jasno označen s `// ZADATAK X`

4. **Ne mijenjaj `index.html`** — svi elementi su već napravljeni

5. Testiraj u Chromeu — otvori `index.html` i provjeri da se sve prikazuje ispravno

6. Commit i push kad završiš:
   ```bash
   git add .
   git commit -m "js-test-vjezba-rijesena"
   git push
   ```
